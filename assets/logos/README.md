# Sports Command Center Logo Assets

Version 0.16.3 added a controlled local logo system.

Recommended folders:

- `channels/` for TV and streaming marks.
- `leagues/` for league, competition, and series marks.
- `teams/` for club, national team, driver, or participant marks.
- `flags/` for national flags.

The app is designed to fall back to styled text badges, sport icons, or flag emoji when a local asset is not mapped yet. Add only assets you have permission to use.
